> 1. 在对话框输入 `@` 呼出文件列表。
> 2. 根据下方表格，选择对应的专家文件。
> 3. 输入指令。

## 命名规则

```
数字_英文缩写_中文备注.md
```

**输入技巧**：
- @00 或 @Basic → 基础聊天
- @01 或 @Task → 任务拆解
- @02 或 @Biz → 商业模式
- @03 或 @ROI → 算账评估
- @04 或 @Course → 课程设计
